## Description
This repository contains two versions of a Spring Boot project connected to a PostgreSQL database:

- Version managed with **Visual Studio Code**
- Version managed with **Apache NetBeans**

Both projects demonstrate a basic Spring Boot application configured to connect to a PostgreSQL database using Spring Data JPA.

- MyFirstDBPry : VisualStudio
- MyFirstDBPry_JD : Apache NetBeans
